# 📅 Cantinho Madalena - Gestão de Turnos

Uma aplicação Web simples (Single-Page Application) concebida para gerir de forma eficiente os horários e folgas da equipa do Cantinho da Madalena.

## 🛠 Funcionalidades
* Vista Semanal: Geração automática da escala de turnos (semanas pares/ímpares).
* Gestão de Exceções: Permite assinalar faltas, baixas médicas ou forçar mudanças de turnos pontuais.
* Motor de Folgas Inteligente: Simula e aplica a 2.ª folga semanal garantindo que não há mais de 2 ausências em simultâneo (dando prioridade ao fim de semana).
* Impressão Ecológica (A4): Botão nativo que formata automaticamente a semana atual para tamanho A4, convertendo o tema escuro num tema branco (poupando tinta de impressão).
* PWA: Pode ser instalado como uma "App" no telemóvel (Adicionar ao Ecrã Inicial).
